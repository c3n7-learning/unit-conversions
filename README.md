## Installation

You can install the package via composer:

```bash
composer require c3n7-learning/unit-conversions
```

## Usage

```php
$skeleton = new C3N7Learning\UnitConversions();
echo $skeleton->echoPhrase('Hello, C3N7Learning!');
```
